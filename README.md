# 🔐 Password Manager (Python CLI)

A command-line Password Manager built using Python that securely stores website/app credentials with master password protection.

This project is part of my Python learning roadmap and focuses on building a real-world application using core Python concepts.

---

## ✨ Features

- ➕ Add new credentials
- 🔍 Search credentials
- 👀 View saved credentials
- ✏️ Update username/password
- 🗑️ Delete credentials
- 🔒 Master Password Protection
- ⏳ Session Timeout Authentication
- 🎲 Random Password Generator
- 💪 Password Strength Checker
- 💾 Persistent JSON Storage

---

## 🚀 How to Run

1. Clone the repository

```bash

```

2. Move into the project directory

```bash
cd password-manager-python
```

3. Run the application

```bash
python main.py
```
